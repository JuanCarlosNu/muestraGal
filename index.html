ROTADOR DE BLOCKS DE MOTORES

En este proyecto se diseñó un rotador de blocks de motor para tareas de control entre procesos de mecanizado. Incluyó cálculos dinámicos para seleccionar mando, cálculos para dimensionado y selección de sistema neumático para movimiento de marco y clampeo y verificación teórica FEM de las piezas sometidas a cargas en dicho dispositivo.
